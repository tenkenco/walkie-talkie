# Walkie CLI

A friendly command-line tool from [Tenken](https://tenken.co) for managing your sessions — create, search, enrich, version, and download them right from your terminal.

## Features

- **Session management** — Create, list, search, enrich, version, and download sessions without leaving the terminal.
- **Multi-platform support** — Works with sessions from Claude, Claude Code, Codex, OpenCode, ChatGPT, and more.
- **Built-in auth** — Simple OAuth login so you're up and running in seconds.
- **Scriptable** — Designed for local automation and CI pipelines alike.

## Quick start

### npm

```bash
npm i -g @tenken/walkie-cli
walkie --version
```


## Usage

```
walkie auth      # login, logout, status
walkie sessions  # upload, list, get, search, enrich, download
walkie config    # manage local settings
walkie billing   # check usage & plans
```

Run `walkie --help` or `walkie <group> --help` for details on any command.
